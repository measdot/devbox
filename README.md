# DevBox


## Installation
```SHELL
vagrant up
```

add alias for easy ssh as described here 
https://stackoverflow.com/a/8967864/2189773
```SHELL
dev() {
    cd /path/to/devbox
    vagrant ssh
}
```

this will add dev alias to ssh into devbox
